

## Set-up

```
git clone https://github.com/dobladov/oerworlmap-reactive.git
cd oerworlmap-reactive
npm install
```

Create a `.env` file with this configuration

```
ELASTICSEARCH_INDEX = elastic
ELASTICSEARCH_URL = https://oerworldmap.org/
```

## Development

```
npm start
```

## Production

```
npm run build
```

## Tests

```
npm run test
```
